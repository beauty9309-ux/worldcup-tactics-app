## 주제

실제 월드컵 데이터를 활용한 동적 인터랙션 웹서비스 구현.
사용자가 축구 감독이 되어 전술을 직접 구성하고 조작할 수 있는 웹서비스.

예시 아이디어:
- 드래그앤드롭으로 선수를 배치하고 포메이션을 구성하는 전술 보드
- 상대팀 데이터를 분석하고 AI가 맞춤 전술을 추천해주는 서비스
- 실제 경기 데이터를 기반으로 "내가 감독이었다면" 시나리오를 시뮬레이션하는 서비스
- 경기 스탯을 보며 교체·전술 변경을 지시하는 인터랙티브 서비스

## 제출물

### 1) 기획서 (~7/27 10:00, PDF)
포함 내용: 서비스 개요, "감독 경험" 설계 의도, 페이지 구성, 핵심 인터랙션 명세, 데이터 활용 방식, 주요 사용 흐름

### 2) 최종 산출물 (~8/3 10:00)
- (필수) 배포 URL — 별도 설치·회원가입·유료결제 없이 브라우저에서 바로 플레이 가능해야 함
- (필수) GitHub 저장소 링크 — 실행 방법, 사용 기술 등 포함 (제출 마감 이후 커밋 시 실격 가능)
- (필수) 시연영상 (유튜브 링크) — 시작 화면, 선수 배치/전술 설정, 핵심 인터랙션, 결과 화면 포함

## 데이터

- 실제 선수/팀/경기 데이터는 더미 데이터로 직접 구성 권장 (JSON 등)
- 특정 연도·대회·선수 구성 제한 없음, 자유 설정 가능
- 외부 API 사용 시 상업적 이용 가능한 무료 라이선스인지 확인 필요, 키 없이 심사자가 확인 가능해야 함
