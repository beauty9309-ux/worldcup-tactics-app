# 위기의 순간, 내가 감독이라면?

**"내가 축구 감독이라면" 월드컵 전술 웹서비스 챌린지 — 기획서**

- 팀명: 월드콘

- 제출일: 2026-07-27

## 1. 서비스 개요

**"위기의 순간, 내가 감독이라면?"**은 2026 FIFA 월드컵 조별예선에서 대한민국이 실제로
0:1로 끌려가기 시작한 그 순간을 그대로 재현하고, 사용자가 감독이 되어 남은 시간 동안
전술을 다시 설계해 역전을 노려보는 인터랙티브 웹서비스입니다.

- 위기 상황(멕시코전 후반 5분 / 남아공전 후반 18분)을 선택하면, 대한민국이 이미 실점해
  0:1로 끌려가기 시작한 바로 그 순간의 실제 라인업·포메이션이 위기 대응 보드에 그대로
  불러와집니다.
- 사용자는 60초 안에 포메이션을 바꾸거나 선수를 교체하며 남은 시간의 전술을 재설계합니다.
- AI 코치가 먼저 추천 전술을 제시하고, 사용자는 이를 따르거나 자신의 판단으로
  다르게 결정할 수 있습니다.
- 결정이 끝나면 역전 승리 확률과 잔여 시간 재시뮬레이션 결과(승/무/패 중 하나), 그리고
  "AI 추천을 따랐다면 어땠을지"에 대한 비교 평가를 확인합니다.

가상의 팀·가상의 위기가 아니라, **"우리가 질 팀이 아니었는데 졌다"는 실제 팬들의
아쉬움을 데이터로 되짚어보는 경험**이라는 점이 이 서비스의 핵심 전제입니다.

---

## 2. "감독 경험" 설계 의도

| 설계 요소 | 의도 |
|---|---|
| 실제 실점 장면 재현 | 가상 시나리오보다 몰입도가 높은 "그 순간, 나라면 어떻게 했을까"라는 감정적 진입점을 제공 |
| 60초 결정 타이머 | 실제 감독이 벤치에서 느끼는 시간 압박과 긴장감을 재현. 시간 초과 시 현재 상태로 자동 확정되어 "결정을 미룰 수 없는" 현실감을 부여 |
| AI 코치 캐릭터 | 확률과 수치만 나열하지 않고, "감독님, 오른쪽 측면이 위험합니다" 식의 대사로 전달해 대화하듯 몰입 |
| AI 추천 vs 감독의 직감 | 이 서비스의 핵심 차별점. AI가 계산한 최적 전술을 그대로 따를지, 감독으로서 자신의 판단을 믿을지 선택하게 하고, 결과 후 두 선택을 비교해 보여줌으로써 "AI와 인간이 협업(또는 경쟁)하는 감독 경험"을 완성 |
| 탭/드래그 동시 지원 | 데스크톱 드래그앤드롭과 모바일 탭-투-스왑을 모두 지원해, 심사 환경(PC/모바일 무관)에서 조작 직관성을 보장 |

## 3. 페이지 구성

```
화면 1. 위기 상황 선택
  ├─ 멕시코전: 후반 5분, 이미 0:1로 끌려가는 상황
  └─ 남아공전: 후반 18분, 하프타임 3교체 이후 이미 0:1로 끌려가는 상황

화면 2. 위기 대응 보드
  ├─ 60초 결정 타이머
  ├─ AI 코치 브리핑 (상황 설명 + 추천 포메이션 + "AI 추천 적용" 버튼)
  ├─ 포메이션 선택 (6종: 3-4-2-1 / 4-2-3-1 / 4-4-2 / 4-3-3 / 3-5-2 / 5-3-2)
  ├─ 전술보드 (대한민국: 탭/드래그로 교체 가능 · 상대팀: 실제 선발 고정 표시)
  └─ 선수 스탯 비교 (임의 두 선수의 6개 능력치 막대 비교)

화면 3. 결과
  ├─ AI 코치의 평가 (AI 추천 vs 감독님의 선택 비교)
  ├─ 역전 승리 확률 + 실제 결과(승/무/패)
  ├─ 재시뮬레이션 최종 스코어
  ├─ 전술 변경 분석 (수치 기반 설명)
  └─ 경기 타임라인
```

## 4. 핵심 인터랙션 명세

| 인터랙션 | 설명 |
|---|---|
| 포메이션 변경 | 드롭다운 선택 시 현재 선수단을 새 포메이션에 자동 재배치(포지션 유사도 기반 대체 로직 포함) |
| 선수 교체 | 슬롯을 탭하거나 드래그 → 다른 슬롯/벤치를 탭하거나 드롭 → 두 선수의 위치가 교체됨. 같은 대상을 다시 탭하면 선택 취소 |
| 60초 카운트다운 | 진입 즉시 시작, 10초 이하부터 경고색(빨강)으로 전환, 0초 도달 시 현재 전술로 자동 결과 산출 |
| AI 추천 적용 | 버튼 클릭 한 번으로 AI가 계산한 최적 포메이션으로 즉시 전환. 이후에도 자유롭게 다시 변경 가능(강제 아님) |
| 선수 스탯 비교 | 드롭다운으로 임의의 두 선수를 선택하면 스피드/슈팅/패스/드리블/수비/피지컬 6개 지표를 막대로 비교 |
| 결과 재시도 | "같은 전술로 다시 시뮬레이션"(같은 선택으로 재추첨) 또는 "전술 다시 짜기"(보드로 복귀, 60초 타이머 리셋) |

## 5. 데이터 활용 방식

이 서비스는 **실제 검증된 사실 데이터**와 **명시적으로 표시된 추정 게임 데이터**를
분리해서 사용합니다.

**① 실제 데이터 (사실관계 — 임의로 지어내지 않음)**
- 대회/라운드/날짜/경기장, 최종 스코어, 득점자, 득점 시각
- 실점 순간의 실제 선발 포메이션 및 라인업(하프타임 교체 반영)
- 선수 실명, 실제 등번호, 실제 포지션
- 출처: 네이버 스포츠(Naver Sports) 경기 기록 API + `openfootball/worldcup.json`
  (오픈소스 2026 월드컵 스쿼드/등번호/경기결과 데이터)로 상호 교차검증

**② 추정 데이터 (게임적 수치 — 공식 레이팅 아님을 명시)**
- 팀 단위 공격력/미드필드/수비력 레이팅
- 선수 개인 종합능력치(overall) 및 세부 스탯(스피드/슈팅/패스/드리블/수비/피지컬)
- 위 수치는 실제 선수의 일반적 명성/포지션을 참고한 게임 디자인상의 추정치이며,
  EA SPORTS FC 등 상업 게임의 공식 레이팅을 사용하지 않았습니다(라이선스 문제 회피).

**③ AI 판단 엔진**
- 학습된 머신러닝 모델이 아니라, 팀 레이팅과 포메이션별 보정치로 산출한 기대 득점을
  포아송(Poisson) 분포에 대입해 **역전 승/무/패 확률**을 계산하는 규칙 기반(rule-based)
  수식 엔진입니다. 실제 시뮬레이션 결과도 이 확률에 따라 승리·무승부·패배 중 하나로
  무작위 산출되어, 전술을 잘 짜도 반드시 역전하는 것은 아닙니다.
- 모든 데이터는 정적 로컬 데이터로 내장되어 있어 **외부 API 키가 필요 없으며**,
  심사자가 별도 인증 없이 바로 확인할 수 있습니다.

## 6. 주요 사용 흐름

1. 사용자가 위기 상황 카드(멕시코전 / 남아공전) 중 하나를 선택한다.
2. AI 코치의 브리핑을 확인하고, 60초 타이머가 시작된다.
3. 포메이션을 바꾸거나 선수를 탭/드래그로 교체한다. 필요하면 "AI 추천 적용"으로
   AI의 제안을 즉시 반영할 수 있다.
4. "이 전술로 결과 산출하기"를 누르거나, 60초가 지나면 현재 상태로 자동 확정된다.
5. 결과 화면에서 AI 추천과 자신의 선택을 비교하고, 역전 승리 확률·실제 승/무/패
   결과·재시뮬레이션 스코어·수치 기반 설명·타임라인을 확인한다.
6. "같은 전술로 다시" 또는 "전술 다시 짜기"로 원하는 만큼 반복해본다.
